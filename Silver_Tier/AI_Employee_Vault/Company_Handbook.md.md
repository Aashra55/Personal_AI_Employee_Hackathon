## Email Handling Rules

1. Always check if the email is unread and important.
2. Save new emails into /Needs_Action as individual markdown files.
3. If a draft reply is needed, generate it under the AI Processing Result section.
4. Suggested Actions:
   - Reply if question or request is present.
   - Forward to relevant party if indicated.
   - Archive after processing.

## Folder Workflow

- /Inbox → all incoming emails monitored by GmailWatcher
- /Needs_Action → emails that require action
- /Done → processed emails after AI skill has been applied

## Decision Priority

1. High Priority → reply ASAP
2. Medium Priority → handle within 24h
3. Low Priority → archive or defer

## Agent Instructions

- Read emails in /Needs_Action
- Analyze content and determine action
- Draft replies if necessary
- Append AI output to the file
- Move processed files to /Done
- Maintain consistent structured Markdown
